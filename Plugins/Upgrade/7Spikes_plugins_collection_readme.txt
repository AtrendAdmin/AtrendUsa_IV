Steps:
1. Backup your existing database
2. If you haven't upgraded your nopCommerce website to version 3.3 already, then follow the instructions in the "nopCommerce_readme.txt".
3. Execute "7Spikes_upgrade.sql" script over your database.
4. Copy all the plugins for 3.3 to the Plugins folder of nopCommerce.
5. Start your site and from the administration - uninstall and then install again the Nop Ajax Filters plugin.
6. If you have made any changes to the plugins in 3.2 then you need to merge them in the new version of the plugins.
